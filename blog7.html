Charts and Visuals
11/5/20

For Project 2, my partner and I are making a budgeting/spending tracker app. It will track purchases and classify them based on a category and a date. For a good user interface, we need visuals to represent our data.

As always, w3schools comes in to save the day with their tutorial on pie charts. It uses a Google API to create the chart and is extremely simple to use. You create an array of data and the rest is done for you. It has good visuals, splits up each category, draws a proportional chart, and shows the percentages of each section. It also includes a color key and hover details.

It has a bunch of defaults but also has customization options. For this pie chart, I chose each color manually. It also has other random customizations like being able to make it 3d. By default, it displays percentages instead of numbers, which we can also change by adding more to the options variable.

Code:
<div id="piechart"></div> <script type="text/javascript" src="https://www.gstatic.com/charts/loader.js"></script> <script type="text/javascript"> google.charts.load('current', {'packages':['corechart']}); google.charts.setOnLoadCallback(drawChart); function drawChart() { var data = google.visualization.arrayToDataTable([ ['Category', 'Cost'], ['Utilities', 80], ['Food', 120], ['Entertainment', 35], ['Transportation', 30], ['Essentials', 30] ]); var options = { 'title':'Spending this month', 'width': 900, 'height': 550, 'colors': ['#6495ed', '#FF5733', '#0cE848', '#980CE8', '#E8E10C'], 'pieSliceText': "value", }; var chart = new google.visualization.PieChart(document.getElementById('piechart')); chart.draw(data, options); } </script> Return to Table of Contents...
