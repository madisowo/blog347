Sorting Attributes
10/8/20

For this week's blog, I wanted to explore types of sorting methods. Google was not my friend for finding resources this time around, only providing ways to sort lists alphabetically or numerically. Eventually I gave up my search and decided I could do this myself (with the help of StackOverflow of course).

After trying a few different things, I came to the conclusion that sorting in JavaScript sucks. This led me to my end idea of making lots of different lists and only displaying one at a time. I did this with the help of a few StackOverflow threads (one, two).

These threads helped provide me with a basis moving forward of how to hide different div objects using JavaScript and how to call a JS method with a "select" element. The next problem came with the execution of this. How do I get the correct list to show? To keep it simple, the single JS method hides every list before setting the selected element to show. This was accomplished by getting the value of the current item and using that to set the correct list to show.

The list is selected with a dropdown menu. You can sort based of different attributes; "A" will show you only items with an "A" in it and so on.
