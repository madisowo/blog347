Light and Dark Themes
9/18/20

Designing a website is crucial to many business' success. This brings about a hot discussion topic: Light mode vs Dark mode. Some studies have collected data that resulting in as much as 95% of people preferering dark mode over light mode. I am one of the 5%.

I was not always this way. In high school, I took two years of graphic design. One of the things I learned is how the readability of white font on a dark background is not as good as black text on a light background. Ever since I learned that, I have been unable to read more than a few words in dark mode. If I read any more than that, I get a similar effect to when you look at a light or the sun and have a dark spot in your vision. It is just like that but with multiple lines of sun spots. While I do have a color inverter extension, not all websites play nicely with this and it doens't always work well.

The following image is from article that talks about the difficulties of dark themes. It describes the white text on the black background to appear to have "halos" around the text, hurting readability.

Knowing this information leads to a valuable question about web development. What do you do? do you anger the masses or hurt the few? Many websites have already adopted toggle options to switch between light and dark modes. This website provides a great intro to switching themes with CSS and Javascript. An example of the code in action can be found here.

Code:
<!DOCTYPE html> <html> <head> <meta name="viewport" content="width=device-width, initial-scale=1"> <style> body { padding: 25px; background-color: white; color: black; font-size: 25px; } .dark-mode { background-color: black; color: white; } </style> </head> <body> <h2>Toggle Dark/Light Mode</h2> <p>Click the button to toggle between dark and light mode for this page.</p> <button onclick="myFunction()">Toggle dark mode</button> <script> function myFunction() { var element = document.body; element.classList.toggle("dark-mode"); } </script> </body> </html> Return to Table of Contents...
