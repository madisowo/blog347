Search Bars
10/1/20

Search bars are an essential part of any website. It lets a user find a page quickly. It may also be easier than the user trying to figure out how to navigate to the desired page. With my project being a game wiki, this is a very valuable tool to help users navigate around.

This article has a nice tutorial on how to get a designed and visually pleasing search bar button into your website. Sadly, it does not work without also adding PHP, which is another factor of web design that I am unfamiliar with. Without PHP, it will give you a file not found exception because the PHP file does not exist.

After reading through the w3 tutorial that they had on PHP, I wouldn't say that I understand it much better and decided to look into similar search features. I found the following search bar on stack overflow that takes the user input and searches Google with it. Hey, if worst comes to worst, Google probably has the answer they were looking for.

Code 1:
<style> /* Style the search field */ form.example input[type=text] { padding: 10px; font-size: 17px; border: 1px solid grey; float: left; width: 80%; background: #f1f1f1; } /* Style the submit button */ form.example button { float: left; width: 20%; padding: 10px; background: #2196F3; color: white; font-size: 17px; border: 1px solid grey; border-left: none; /* Prevent double borders */ cursor: pointer; } form.example button:hover { background: #0b7dda; } /* Clear floats */ form.example::after { content: ""; clear: both; display: table; } </style> <!-- Load icon library --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css"> <!-- The form --> <form class="example" action="action_page.php"> <input type="text" placeholder="Search.." name="search"> <button type="submit"><i class="fa fa-search"></i></button> </form>
Code 2:
